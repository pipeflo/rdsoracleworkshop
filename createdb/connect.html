Connect to your Oracle Database

In this section you will connect to the RDS for Oracle Instance deployed on the last section.

  1. Once your Database status is Active, click on the database name to view the details.

  2. Now copy the endpoint generated for the database, this will be the hostname you will use in the next step to connect to the database, and it would be the one you would need to use to connect from an application.

  3. Now open SQL Developer, select New Connection and provide the following inputs and then click connect:

    Note: SQL Developer can be downloaded here.

    • Connection name: - < User defined connection name> rdsoracle
    • Username: - Master username provided during Database creation admin
    • Password: - Password provided for the Master Username during Database creation
    • Hostname: - Endpoint copied in the step #2 above
    • SID: - Database name provided during Database Creation MFDB

    Now you can see your connection to the Oracle database on the connections tab of SQL Developer.

  4. If the connection did not work you can try updating the Security Group created, following the next steps, if it did work, continue to step 5.

    • Return to the RDS Console and on the RDS instance details, click on the link for the Oracle security group found on the Security column.
    • On the security group details, click on the Inbound rules tab and then click the Edit inbound rules button
    • On the Inbound rules for the Oracle Security group, click the “Source” option for the rule created, and change the selection from “My IP” to “Anywhere” then click on Save rules
    • After changing the Security Group rule, test again the connection to you RDS for Oracle instance from SQL Developer

  5. Now we are going to connect to the Instance using SQL*PLUS.

    • Launch SQL*PLUS client and use the following command to connect to the DB, replace the host name with the your instance endpoint and SID with the your DB name.
    sqlplus '<username>@(DESCRIPTION=(ADDRESS=(PROTOCOL=TCP)(HOST=<Hostname>)(PORT=1521))(CONNECT_DATA=(SID=<SID>)))'

    • Now enter your password and finally enter the following command:
    select name from v$database;
