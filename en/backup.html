Backup and Restore

In this chapter you will use the diferent options provided by RDS to Backup and Restore your Oracle Instance.

Automated Backup

  • Scheduled daily backup of entire instance
  • Up to 35 day retention for backups
  • Point-in-Time Recovery to any point in time during the backup retention period

Manual Snapshots

  • You can also back up your DB instance manually, by manually creating a DB snapshot
  • Can be used to create a new RDS instance

Transaction logs

RDS uploads transaction logs for DB instances to Amazon S3 every 5 minutes
