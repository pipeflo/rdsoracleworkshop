Export RDS Oracle logs to CloudWatch

To view the detailed logs and performance insights of your RDS Oracle Instance, follow the next steps:

  1. Find the RDS Oracle created instance under the Databases list and click on the DB Instance name.
  2. On the RDS Instance details, click on the Modify button.
  3. On the Modify DB Instance page make the following changes:
    • On the “Monitoring” section select the Enable enhanced monitoring option.
    • On “Log Exports” section select the logs which you would like to get publish on Cloudwatch and click continue-
    • On the Performance Insights section select the Enable Performance Insights option.
